# Product preview card
Responsive Product Preview Card Component from Frontend Mentor

## Overview

This is a **Product Preview Card** project from Frontend Mentor.  
It shows a product with image, title, description, price, and a button.  

It works on both **mobile** and **desktop** screens.

## What’s Inside

- Product image (changes for mobile and desktop)  
- Product category, title, and description  
- Current and original price  
- "Add to Cart" button with hover effect  

## Built With

- HTML  
- CSS  
- Flexbox  
- Mobile-first design  

## Style Guide

- **Mobile width:** 375px  
- **Desktop width:** 1440px  

**Colors:**  
- Green, Dark Green, Black, Grey, Cream, White  

**Fonts:**  
- Montserrat (normal & bold)  
- Fraunces (bold)  
- Paragraph size: 14px  

## Features

- Works on mobile and desktop  
- Hover effect on button  
- Simple, clean layout
